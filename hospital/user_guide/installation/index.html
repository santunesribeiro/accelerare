Instruções de Instalação¶

CodeIgniter é instalado em 4 passos:

  1. Descompacte o arquivo ZIP.
  2. Faça o Upload das pastas e arquivos do CodeIgniter para o seu servidor. Normalmente o arquivo index.php estará no seu diretório raiz.
  3. Abra o arquivo application/config/config.php com um editor de texto e defina a sua URL base. Se você pretende usar criptografia ou sessões, defina a sua chave de criptografia.
  4. Se você pretende usar banco de dados, abra o arquivo application/config/database.php com um editor de texto e defina as suas configurações de banco de dados.

Se você deseja aumentar a segurança escondendo a localização dos seus arquivos do CodeIgniter, você pode renomear as pastas "system" e "application" para algum nome mais privado. Se você renomeá-las, você deve abrir o seu principal arquivo index.php e alterar as variáveis $system_path e $application_folder no topo do arquivo para o nome que você tiver escolhido.

Para maior segurança, ambas as pastas "system" e "application" devem ser colocadas acima do diretório web raiz para não serem acessíveis diretamente via navegador. Por padrão, o arquivo .htaccess está incluso em cada pasta para ajudar a prevenir o acesso direto, mas é melhor removê-los completamente do acesso público no caso de as alterações de configuração de servidor web não respeitar o.htaccess.

Se você desejar manter suas "views" públicas, é também possível mover as pastas views para fora da pasta application.

Depois de movê-las, abra o seu principal arquivo index.php e modifique as variáveis $system_path, $application_folder e $view_folder, preferencialmente com o caminho completo, e.g. ‘/www/MyUser/system‘.

Uma medida adicional a se considerar em ambientes de produção é desabilitar o relatório de erro do PHP ou qualquer outra funcionalidade de desenvolvimento. No CodeIgniter, isso pode ser feito configurando a constante ENVIRONMENT, que é melhor descrita na página de segurança.

E é isso!

Se você é novo no CodeIgniter, por favor, leia a seção Primeiros Passos do Guia de Usuário para começar aprendendo como construir aplicações dinâmicas em PHP. Aproveite!
