Classe de Carrinho de Compras¶

A Classe do carrinho permite que itens sejam adicionados a uma sessão que permaneça ativa enquanto o usuário estiver navegando em seu site. Esses itens podem ser recuperados e exibidos em um formato padrão de "carrinho de compras", permitindo ao usuário atualizar a quantidade ou remover itens do carrinho.

Importante

A biblioteca do carrinho é DEPRECIADA e não deve ser usada. Atualmente, ele é mantido somente para compatibilidade com versões anteriores.

Por favor, note que a Classe de carrinho APENAS fornece a funcionalidade principal do "carrinho". Não fornece envio, autorização de cartão de crédito ou outros componentes de processamento.

Usando a Classe Carrinho de Compra¶

Inicializando a Classe de Carrinho de Compras¶

Importante

A classe do carrinho utiliza a Classe de Sessão do CodeIgniter para salvar as informações do carrinho em um banco de dados, então, antes de usar a classe de carrinho, você deve configurar uma tabela de banco de dados conforme indicado na Documentação da Sessão e definir as preferências da sessão na arquivo aplicação/config/config.php para utilizar um banco de dados.

Para inicializar a Classe de Carrinho de Compras no construtor do seu controlador, use o método $this->load->library():

$this->load->library('cart');

Uma vez carregado, o objeto Carrinho estará disponível usando:

$this->cart

Nota

A Classe de carrinho irá carregar e inicializar a Classe de sessão automaticamente, então, a menos que você esteja usando sessões em outro lugar do seu aplicativo, não é necessário carregar a classe da sessão.

Adicionando Itens ao Carrinho¶

Para adicionar um item ao carrinho de compras, basta passar uma matriz com as informações do produto para o método $this->cart->insert() como mostrado abaixo:

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'options' => array('Size' => 'L', 'Color' => 'Red')
);

$this->cart->insert($data);

Importante

Os primeiros quatro índices de matriz acima (id, qty, preço e nome) são obrigatórios. Se você omitir algum deles, os dados não serão salvos no carrinho. O quinto índice (opções) é opcional. Ele se destina a ser usado nos casos em que seu produto possui opções associadas a ele. Use uma matriz para opções, como mostrado acima.

Os cinco índices reservados são:

  • id - Cada produto em sua loja deve ter um identificador exclusivo. Normalmente, isso será um "sku" ou outro identificador desse tipo.
  • qty - A quantidade que está sendo comprada.
  • price - O preço do item.
  • name - O nome do item.
  • options - Quaisquer atributos adicionais que são necessários para identificar o produto. Estes devem ser transmitidos através de uma matriz.

Além dos cinco índices acima, existem duas palavras reservadas: rowid e subtotal. Estes são usados ​​internamente pela classe Cart, então NÃO use essas palavras como nomes de índice ao inserir dados no carrinho.

Sua matriz pode conter dados adicionais. Qualquer coisa que você incluir na sua matriz será armazenada na sessão. No entanto, é melhor padronizar seus dados entre todos os seus produtos para facilitar a exibição das informações em uma tabela.

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'coupon'         => 'XMAS-50OFF'
);

$this->cart->insert($data);

O método insert() irá retornar o $rowid se você inserir com sucesso um único item.

Adicionando vários itens ao carrinho¶

Ao usar uma matriz multidimensional, como mostrado abaixo, é possível adicionar vários produtos ao carrinho em uma ação. Isso é útil nos casos em que você deseja permitir que pessoas selecionem entre vários itens na mesma página.

$data = array(
        array(
                'id'      => 'sku_123ABC',
                'qty'     => 1,
                'price'   => 39.95,
                'name'    => 'T-Shirt',
                'options' => array('Size' => 'L', 'Color' => 'Red')
        ),
        array(
                'id'      => 'sku_567ZYX',
                'qty'     => 1,
                'price'   => 9.95,
                'name'    => 'Coffee Mug'
        ),
        array(
                'id'      => 'sku_965QRS',
                'qty'     => 1,
                'price'   => 29.95,
                'name'    => 'Shot Glass'
        )
);

$this->cart->insert($data);

Mostrando o Carrinho¶

Para exibir o carrinho, você criará um arquivo de view com um código semelhante ao mostrado abaixo.

Observe que este exemplo usa o auxiliar de formulário.

<?php echo form_open('path/to/controller/update/method'); ?>

<table cellpadding="6" cellspacing="1" style="width:100%" border="0">

<tr>
        <th>QTY</th>
        <th>Item Description</th>
        <th style="text-align:right">Item Price</th>
        <th style="text-align:right">Sub-Total</th>
</tr>

<?php $i = 1; ?>

<?php foreach ($this->cart->contents() as $items): ?>

        <?php echo form_hidden($i.'[rowid]', $items['rowid']); ?>

        <tr>
                <td><?php echo form_input(array('name' => $i.'[qty]', 'value' => $items['qty'], 'maxlength' => '3', 'size' => '5')); ?></td>
                <td>
                        <?php echo $items['name']; ?>

                        <?php if ($this->cart->has_options($items['rowid']) == TRUE): ?>

                                <p>
                                        <?php foreach ($this->cart->product_options($items['rowid']) as $option_name => $option_value): ?>

                                                <strong><?php echo $option_name; ?>:</strong> <?php echo $option_value; ?><br />

                                        <?php endforeach; ?>
                                </p>

                        <?php endif; ?>

                </td>
                <td style="text-align:right"><?php echo $this->cart->format_number($items['price']); ?></td>
                <td style="text-align:right">$<?php echo $this->cart->format_number($items['subtotal']); ?></td>
        </tr>

<?php $i++; ?>

<?php endforeach; ?>

<tr>
        <td colspan="2"> </td>
        <td class="right"><strong>Total</strong></td>
        <td class="right">$<?php echo $this->cart->format_number($this->cart->total()); ?></td>
</tr>

</table>

<p><?php echo form_submit('', 'Update your Cart'); ?></p>

Atualizando o Carrinho¶

Para atualizar as informações em seu carrinho, você deve passar uma matriz contendo o ID da linha e uma ou mais propriedades pré-definidas para o método $this->cart->update().

Nota

Se a quantidade for definida como zero, o item será removido do carrinho.

$data = array(
        'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'   => 3
);

$this->cart->update($data);

// Ou uma matriz multidimensional

$data = array(
        array(
                'rowid'   => 'b99ccdf16028f015540f341130b6d8ec',
                'qty'     => 3
        ),
        array(
                'rowid'   => 'xw82g9q3r495893iajdh473990rikw23',
                'qty'     => 4
        ),
        array(
                'rowid'   => 'fh4kdkkkaoe30njgoe92rkdkkobec333',
                'qty'     => 2
        )
);

$this->cart->update($data);

Você também pode atualizar qualquer propriedade que você definiu anteriormente ao inserir o item, como opções, preço ou outros campos personalizados.

$data = array(
        'rowid'  => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'    => 1,
        'price'  => 49.95,
        'coupon' => NULL
);

$this->cart->update($data);

O que é o Row ID?¶

O row ID é um identificador exclusivo que é gerado pelo código do carrinho quando um item é adicionado ao carrinho. O motivo pelo qual uma ID única é criada é para que produtos idênticos com diferentes opções possam ser gerenciados pelo carrinho.

Por exemplo, digamos que alguém compra duas camisas idênticas (ID do mesmo produto), mas em tamanhos diferentes. A identificação do produto (e outros atributos) será idêntica para ambos os tamanhos porque é a mesma camisa. A única diferença será o tamanho. O carrinho deve, portanto, ter um meio de identificar essa diferença para que os dois tamanhos de camisas possam ser gerenciados de forma independente. Ele faz isso criando um único "Row ID" com base no ID do produto e quaisquer opções associadas a ele.

Em quase todos os casos, a atualização do carrinho será algo que o usuário faça através da página "carrinho de exibição", de modo que, como desenvolvedor, é improvável que você tenha que se preocupar com a "ID da linha", além de se certificar de sua página "visualizar carrinho" contém essas informações em um campo de formulário oculto e certificando-se de que é passado para o método update() quando o formulário de atualização é enviado. Por favor, examine a construção da página "carrinho de exibição" acima para obter mais informações.

Class Referência de Classe¶

class CI_Cart¶
$product_id_rules = '.a-z0-9_-'

Estas são as regras de expressão regular que usamos para validar o ID do produto - alfa-numérico, traços, sublinhados ou períodos por padrão.

$product_name_rules = 'w -.:'

Estas são as regras de expressão regulares que usamos para validar o ID do produto e o nome do produto - alfa-numérico, traços, sublinhados, dois pontos ou períodos por padrão.

$product_name_safe = TRUE

Independentemente de permitir ou não apenas nomes seguros de produtos. Padrão TRUE.

insert([$items = array()])¶
Parâmetros:
  • $items (array) – Itens para inserir no carrinho
Retorna:

TRUE em caso de sucesso, FALSE em falha
Tipo de retorno:

booleano

Insira itens no carrinho e salve-o na tabela de sessão. Retorna TRUE em caso de sucesso e FALSE em caso de falha.

update([$items = array()])¶
Parâmetros:
  • $items (array) – Itens para atualizar no carrinho
Retorna:

TRUE em caso de sucesso, FALSE em falha
Tipo de retorno:

booleano

Este método permite alterar as propriedades de um determinado item. Normalmente, ele é chamado a partir da página "carrinho" se um usuário fizer alterações na quantidade antes da compra. Essa matriz deve conter o rowid para cada item.

remove($rowid)¶
Parâmetros:
  • $rowid (int) – ID do item a ser removido do carrinho
Retorna:

TRUE em caso de sucesso, FALSE em falha
Tipo de retorno:

booleano

Permite que você remova um item do carrinho de compras passando o $rowid.

total()¶
Retorna:Valor total
Tipo de retorno:int

Mostra a quantidade total no carrinho.

total_items()¶
Retorna:Quantidade total de itens no carrinho
Tipo de retorno:int

Mostra o número total de itens no carrinho.

contents([$newest_first = FALSE])¶
Parâmetros:
  • $newest_first (bool) – Se deseja ordenar a matriz com os itens mais novos primeiro
Retorna:

Array do conteúdo do carrinho
Tipo de retorno:

array

Retorna uma matriz contendo tudo no carrinho. Você pode ordenar a ordem pela qual a matriz é retornada passando-a VERDADEIRA, onde o conteúdo será ordenado de mais novo para o mais antigo, caso contrário, ele é ordenado de mais antigo para o mais novo.

get_item($row_id)¶
Parâmetros:
  • $row_id (int) – Row ID para recuperar
Retorna:

Array de itens
Tipo de retorno:

array

Retorna uma matriz contendo dados para o item que corresponde à ID da linha especificada, ou FALSE se nenhum desses itens existir.

has_options($row_id = '')¶
Parâmetros:
  • $row_id (int) – Row ID para inspecionar
Retorna:

TRUE se a opção for verdadeira, ou FALSE para outra forma
Tipo de retorno:

booleano

Retorna TRUE (booleano) se uma determinada linha no carrinho contiver opções. Este método foi projetado para ser usado em um loop com contents(), uma vez que você deve passar o rowid para este método, conforme mostrado no exemplo "Mostrando o Carrinho" acima.

product_options([$row_id = ''])¶
Parâmetros:
  • $row_id (int) – Row ID
Retorna:

Array das opções do produto
Tipo de retorno:

array

Retorna uma série de opções para um produto específico. Este método foi projetado para ser usado em um loop com contents(), uma vez que você deve passar o rowid para este método, conforme mostrado no exemplo Mostrar o carrinho acima.

destroy()¶
Tipo de retorno:void

Permite que você destrua o carrinho. Este método provavelmente será chamado quando você terminar de processar a ordem do cliente.
