Auxiliar de HTML¶

O arquivo Auxiliar de HTML contêm funções que ajudam a trabalhar com HTML.

  • Carregando este Auxiliar
  • Funções disponíveis

Carregando este Auxiliar¶

Este auxiliar é carregado usando o seguinte código:

$this->load->helper('html');

Funções disponíveis¶

As seguinte funções estão disponíveis:

heading([$data = ''[, $h = '1'[, $attributes = '']]])¶
Parâmetros:
  • $data (string) – Conteúdo
  • $h (string) – Nível do Heading
  • $attributes (misto) – atributos HTML
Retorna:

Tag heading do HTML
Tipo de retorno:

string

Permite que você crie tags heading do HTML. O primeiro parâmetro vai conter o título, o segundo o nível do heading. Exemplo:

echo heading('Welcome!', 3);

O código acima vai produzir: <h3>Welcome!</h3>

Adicionalmente, em ordem para adicionar atributos para a tag heading como uma classe HTML, ids ou estilos em linha, um terceiro parâmetro aceita tanto uma string como um array:

echo heading('Welcome!', 3, 'class="pink"');
echo heading('How are you?', 4, array('id' => 'question', 'class' => 'green'));

O código acima produz:

<h3 class="pink">Welcome!<h3>
<h4 id="question" class="green">How are you?</h4>
img([$src = ''[, $index_page = FALSE[, $attributes = '']]])¶
Parâmetros:
  • $src (string) – Local de origem da imagem
  • $index_page (bool) – Determina se o $src será tratado como uma string URI roteada
  • $attributes (array) –atributos HTML
Retorna:

Tag imagem HTML
Tipo de retorno:

string

Permite que você crie tag HTMLL <img />. O primeiro parâmetro contêm a origem da imagem. Exemplo:

echo img('images/picture.jpg'); // produz <img src="http://site.com/images/picture.jpg" />

Existem um segundo parâmetro opcional que é um valor TRUE/FALSE que especifica se o src deve ter a página especificada pelo $config['index_page'] adicionado ao endereço que é criado. Presumivelmente, isto deveria ser feito se você estivesse utilizando um controlador de mídia:

echo img('images/picture.jpg', TRUE); // produz <img src="http://site.com/index.php/images/picture.jpg" alt="" />

Adicionalmente, um array associativo pode ser passado para a função img() para total controle sobre todos os atributos e valores. Se um atributo alt não é provido, o CodeIgniter vai gerar uma string vazia.

Exemplo:

$propriedades_da_imagem = array(
        'src'   => 'images/picture.jpg',
        'alt'   => 'Eu, demostrando como comer 4 pedaços de pizza de uma única vez',
        'class' => 'post_images',
        'width' => '200',
        'height'=> '200',
        'title' => 'Foi uma noite daquelas',
        'rel'   => 'lightbox'
);

img($propriedades_da_imagem);
// <img src="http://site.com/index.php/images/picture.jpg" alt="Eu, demostrando como comer 4 pedaços de pizza de uma única vez" class="post_images" width="200" height="200" title="Foi uma noite daquelas" rel="lightbox" />
link_tag([$href = ''[, $rel = 'stylesheet'[, $type = 'text/css'[, $title = ''[, $media = ''[, $index_page = FALSE]]]]]])¶
Parâmetros:
  • $href (string) – Para onde estamos apontando
  • $rel (string) – Tipo de Relação
  • $type (string) – Tipo de um documento relacionado
  • $title (string) – Título do Link
  • $media (string) – Tipo da mídia
  • $index_page (bool) – Determina se $src deverá ser tratado como uma string da URI roteada
Retorna:

Tag link do HTML
Tipo de retorno:

string

Permite que você crie tags <link /> do HTML. Isto é tão útil para links de folhas de estilo, como para outros links. Os parâmetros são href, tendo como opcionais rel, type, title, media e index_page.

index_page é um valor boleano que determina se o href deve ter a página especificada pelo $config['index_page'] adicionado ao endereço que ele cria.

Exemplo:

echo link_tag('css/mystyles.css');
// produz <link href="http://site.com/css/mystyles.css" rel="stylesheet" type="text/css" />

Outros exemplos:

echo link_tag('favicon.ico', 'shortcut icon', 'image/ico');
// <link href="http://site.com/favicon.ico" rel="shortcut icon" type="image/ico" />

echo link_tag('feed', 'alternate', 'application/rss+xml', 'My RSS Feed');
// <link href="http://site.com/feed" rel="alternate" type="application/rss+xml" title="My RSS Feed" />

Adicionalmente, um array associativo pode ser passado para a funçãolink() para controle completo sobre todos os atributos e valores:

$link = array(
        'href'  => 'css/printer.css',
        'rel'   => 'stylesheet',
        'type'  => 'text/css',
        'media' => 'print'
);

echo link_tag($link);
// <link href="http://site.com/css/printer.css" rel="stylesheet" type="text/css" media="print" />
ul($list[, $attributes = ''])¶
Parâmetros:
  • $list (array) – Entradas da lista
  • $attributes (array) – HTML atributos
Retorna:

Lista HTML não ordenada formatada
Tipo de retorno:

string

Permite que vocẽ gere listas HTML não ordenadas de um array simples ou multidimensional. Exemplo:

$list = array(
        'red',
        'blue',
        'green',
        'yellow'
);

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

echo ul($list, $attributes);

O código acima vai produzir isto:

<ul class="boldlist" id="mylist">
        <li>red</li>
        <li>blue</li>
        <li>green</li>
        <li>yellow</li>
</ul>

Aqui está um exemplo mais complexo, usando um array multidimensional:

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

$list = array(
        'colors'  => array(
                'red',
                'blue',
                'green'
        ),
        'shapes'  => array(
                'round',
                'square',
                'circles' => array(
                        'ellipse',
                        'oval',
                        'sphere'
                )
        ),
        'moods'  => array(
                'happy',
                'upset' => array(
                        'defeated' => array(
                                'dejected',
                                'disheartened',
                                'depressed'
                        ),
                        'annoyed',
                        'cross',
                        'angry'
                )
        )
);

echo ul($list, $attributes);

O código acima irá produzir isto:

<ul class="boldlist" id="mylist">
        <li>colors
                <ul>
                        <li>red</li>
                        <li>blue</li>
                        <li>green</li>
                </ul>
        </li>
        <li>shapes
                <ul>
                        <li>round</li>
                        <li>suare</li>
                        <li>circles
                                <ul>
                                        <li>elipse</li>
                                        <li>oval</li>
                                        <li>sphere</li>
                                </ul>
                        </li>
                </ul>
        </li>
        <li>moods
                <ul>
                        <li>happy</li>
                        <li>upset
                                <ul>
                                        <li>defeated
                                                <ul>
                                                        <li>dejected</li>
                                                        <li>disheartened</li>
                                                        <li>depressed</li>
                                                </ul>
                                        </li>
                                        <li>annoyed</li>
                                        <li>cross</li>
                                        <li>angry</li>
                                </ul>
                        </li>
                </ul>
        </li>
</ul>
ol($list, $attributes = '')¶
Parâmetros:
  • $list (array) – Entradas da lista
  • $attributes (array) – Atributos HTML
Retorna:

Lista ordenada HTML formatada
Tipo de retorno:

string

Idêntico ao ul(), porém será produzido a tag <ol> para listas ordenadas ao invés de <ul>.

meta([$name = ''[, $content = ''[, $type = 'name'[, $newline = "n"]]]])¶
Parâmetros:
  • $name (string) – Nome da tag meta
  • $content (string) – Conteúdo da tag meta
  • $type (string) – Tipo da tag meta
  • $newline (string) – Caractere nova linha
Retorna:

Tag meta HTML
Tipo de retorno:

string

Auxilia você a gerar tags meta. Você pode passar strings para a função, array simples ou arrays multidimensionais.

Exemplos:

echo meta('description', 'Meu grande site');
// Gera:  <meta name="description" content="Meu grande site" />

echo meta('Content-type', 'text/html; charset=utf-8', 'equiv');
// Note o terceiro parãmetro. Ele pode ser "equiv" ou "name"
// Gera:  <meta http-equiv="Content-type" content="text/html; charset=utf-8" />

echo meta(array('name' => 'robots', 'content' => 'no-cache'));
// Gera:  <meta name="robots" content="no-cache" />

$meta = array(
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'description',
                'content' => 'Meu grande site'
        ),
        array(
                'name' => 'keywords',
                'content' => 'amor, paixão, intriga, decepção'
        ),
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'Content-type',
                'content' => 'text/html; charset=utf-8', 'type' => 'equiv'
        )
);

echo meta($meta);
// Gera:
// <meta name="robots" content="no-cache" />
// <meta name="description" content="Meu grande site" />
// <meta name="keywords" content="amor, paixão, intriga, decepção" />
// <meta name="robots" content="no-cache" />
// <meta http-equiv="Content-type" content="text/html; charset=utf-8" />
doctype([$type = 'xhtml1-strict'])¶
Generates
Parâmetros:
  • $type (string) – Nome do tipo do documento
Retorna:

Tag HTML DocType
Tipo de retorno:

string

Auxilia você a gerar a declaração do tipo do documento, ou DTD’s. XHTML 1.0 Strict é usado por padrão, mas muitos doctypes estão disponíveis.

Exemplo:

echo doctype(); // <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

echo doctype('html4-trans'); // <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

br([$count = 1])¶
Parâmetros:
  • $count (int) – Número de vezes que a tag será repetida
Retorna:

Tag HTML de quebra de linha
Tipo de retorno:

string

Gera as tags (<br />) para quebra de linha baseado no número que foi informado no parâmetro. Exemplo:

echo br(3);

O código acima irá produzir:

<br /><br /><br />

Nota

Esta função está OBSOLETA. Use a função nativa str_repeat() em conjunto com <br /> como alternativa.

nbs([$num = 1])¶
Parâmetros:
  • $num (int) – Número de entidades de espaços a serem produzidas
Retorna:

Uma sequência de entidades de espaços HTML não quebráveis
Tipo de retorno:

string

Gera não quebráveis espaços (&nbsp;) baseados no número que você informar no parâmetro. Exemplo:

echo nbs(3);

O código acima irá produzir:

&nbsp;&nbsp;&nbsp;

Nota

Esta função está OBSOLETA. Use a função nativa str_repeat() em conjunto com &nbsp; como alternativa.
