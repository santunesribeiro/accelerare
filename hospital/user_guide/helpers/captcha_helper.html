Auxiliar de CAPTCHA¶

O arquivo Auxiliar de CAPTCHA contém funções que auxiliam na criação de imagens CAPTCHA.

  • Carregando este Auxiliar
  • Usando o Auxiliar de CAPTCHA
    • Adicionando um Banco de Dados
  • Funções disponíveis

Carregando este Auxiliar¶

Este auxiliar é carregado usando o seguinte código:

$this->load->helper('captcha');

Usando o Auxiliar de CAPTCHA¶

Uma vez carregado você pode gerar um CAPTCHA como este:

$vals = array(
        'word'          => 'Random word',
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/',
        'font_path'     => './path/to/fonts/texb.ttf',
        'img_width'     => '150',
        'img_height'    => 30,
        'expiration'    => 7200,
        'word_length'   => 8,
        'font_size'     => 16,
        'img_id'        => 'Imageid',
        'pool'          => '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ',

        // Fundo e borda brancos, texto preto e grade vermelha
        'colors'        => array(
                'background' => array(255, 255, 255),
                'border' => array(255, 255, 255),
                'text' => array(0, 0, 0),
                'grid' => array(255, 40, 40)
        )
);

$cap = create_captcha($vals);
echo $cap['image'];
  • A função captcha requer a biblioteca de imagens GD.
  • Somente são requeridos os parametros img_path e img_url.
  • Se o parametro word não for fornecido, a função irá gerar uma string ASCII aleatória. Você pode juntar sua própria biblioteca de palavras da qual você pode desenhar aleatoriamente.
  • Se você não especificar um caminho para uma fonte TRUE TYPE, a fonte nativa GD feia será usada.
  • O diretório "captcha" deve ser gravável
  • A expiration (em segundos) significa quanto tempo uma imagem permanecerá na pasta captcha antes de ser apagada. O padrão é duas horas.
  • O padrão de word_length é 8, o padrão do pool é '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'
  • O padrão de font_size é 16, a fonte nativa GD tem um limite de tamanho. Especifique uma fonte "true type" para tamanhos maiores.
  • O img_id será definido como o "id" da imagem captcha.
  • Se algum dos valores de colors estiver faltando, ele será substituído pelo padrão.

Adicionando um Banco de Dados¶

Para que a função captcha evite que alguém envie, você precisará adicionar as informações retornadas de create_captcha() ao seu banco de dados. Então, quando os dados do formulário forem enviados pelo usuário, você precisará verificar se os dados existem no banco de dados e não expirou.

Aqui está um protótipo de tabela:

CREATE TABLE captcha (
        captcha_id bigint(13) unsigned NOT NULL auto_increment,
        captcha_time int(10) unsigned NOT NULL,
        ip_address varchar(45) NOT NULL,
        word varchar(20) NOT NULL,
        PRIMARY KEY `captcha_id` (`captcha_id`),
        KEY `word` (`word`)
);

Aqui está um exemplo de uso com um banco de dados. Na página em que o CAPTCHA será mostrado, você terá algo como isto:

$this->load->helper('captcha');
$vals = array(
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/'
);

$cap = create_captcha($vals);
$data = array(
        'captcha_time'  => $cap['time'],
        'ip_address'    => $this->input->ip_address(),
        'word'          => $cap['word']
);

$query = $this->db->insert_string('captcha', $data);
$this->db->query($query);

echo 'Envie a palavra que vê abaixo:';
echo $cap['image'];
echo '<input type="text" name="captcha" value="" />';

Então, na página que aceita a submissão, você terá algo como isto:

// Primeiro, apague os captchas antigos
$expiration = time() - 7200; // Limite de duas horas
$this->db->where('captcha_time < ', $expiration)
        ->delete('captcha');

// Veja então se existe um captcha:
$sql = 'SELECT COUNT(*) AS count FROM captcha WHERE word = ? AND ip_address = ? AND captcha_time > ?';
$binds = array($_POST['captcha'], $this->input->ip_address(), $expiration);
$query = $this->db->query($sql, $binds);
$row = $query->row();

if ($row->count == 0)
{
        echo 'Você deve enviar a palavra que aparece na imagem.';
}

Funções disponíveis¶

As seguinte funções estão disponíveis:

create_captcha([$data = ''[, $img_path = ''[, $img_url = ''[, $font_path = '']]]])¶
Parâmetros:
  • $data (array) – Array de dados para o CAPTCHA/li>
  • $img_path (string) – Caminho para criar a imagem
  • $img_url (string) – URL para a pasta de imagem CAPTCHA
  • $font_path (string) – Caminho do servidor para a fonte
Retorna:

array(‘word’ => $word, ‘time’ => $now, ‘image’ => $img)
Tipo de retorno:

array

Toma um array de informações para gerar o CAPTCHA como entrada e cria a imagem para suas especificações, retornando um array de dados associativos sobre a imagem.

array(
        'image' => IMAGE TAG
        'time'  => TIMESTAMP (in microtime)
        'word'  => CAPTCHA WORD
)

A image é a marca de imagem atual:

<img src="http://example.com/captcha/12345.jpg" width="140" height="50" />

O time é o timestamp micro usado como o nome da imagem sem a extensão do arquivo. Será um número como este: 1139612155.3422

O word é a palavra que aparece na imagem captcha, que, se não for fornecida com a função, será uma string aleatória.
