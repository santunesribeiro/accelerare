Auxiliar de Texto¶

O arquivo Auxiliar de Texto contêm funções que auxiliam no trabalho com texto.

  • Carregando este Auxiliar
  • Funções disponíveis

Carregando este Auxiliar¶

Este auxiliar é carregado usando o seguinte código:

$this->load->helper('text');

Funções disponíveis¶

As seguinte funções estão disponíveis:

word_limiter($str[, $limit = 100[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) – String de entrada
  • $limit (int) – Limite
  • $end_char (string) – Caractere final (geralmente uma reticência)
Retorna:

String com limite de palavras
Tipo de retorno:

string

Trunca uma string com o número de palavras especificadas. Exemplo:

$string = "Aqui tem um bom texto string composto com dez palavras.";
$string = word_limiter($string, 4);
// Retorna:  Aqui tem um bom

O terceiro parâmetro é um sufixo opcional adicionado à string. Por padrão é adicionado uma reticência.

character_limiter($str[, $n = 500[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) – String de entrada
  • $n (int) – Número de caracteres
  • $end_char (string) – Caractere final (geralmente uma reticência)
Retorna:

String com limite de caracteres
Tipo de retorno:

string

Trunca uma string com o número de caracteres especificados. Ele mantém a integridade das palavras, então a quantidade de caracateres pode ser um pouco diferente de sua especificação.

Exemplo:

$string = "Aqui tem um bom texto string composto de dez palavras.";
$string = character_limiter($string, 20);
// Retorna:  Aqui tem um bom texto string

O terceiro parâmetro é um sufixo opcional para a string. Se não for declarado o ajudante usará uma reticência.

Nota

Se você precisar truncar em um exato número de caracteres por favor veja a função ellipsize() abaixo.

ascii_to_entities($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

Uma string com valores ASCII convertida para entidades
Tipo de retorno:

string

Converte valores ASCII para entidades de caracter, incluindo o alto ASCII e caracteres do MS Word que podem causar problemas quando usados em uma página da web. Dessa forma, eles poderão ser mostrados de forma consistente, independente das configurações do navegador ou serem armazenados de forma confiável em um banco de dados. Existem algumas dependências dos conjuntos de caracteres suportados pelo seu servidor, portanto pode ser que a função não funcione 100% em todos os casos, mas na maioria das vezes ela identificará corretamente os caracteres fora do normal (como caracteres com acentos).

Exemplo:

$string = ascii_to_entities($string);
convert_accented_characters($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

Uma string com os caracteres acentuados convertidos
Tipo de retorno:

string

Converte caracteres em alto ASCII para os seus correspondentes em baixo ASCII. Utilizado quando caracteres não ingleses precisam ser utilizados onde somente caracteres ASCII padrão são permitidos, como por exemplo, em URLs.

Exemplo:

$string = convert_accented_characters($string);

Nota

Essa função utiliza um arquivo de configuração complementar application/config/foreign_chars.php para definir o array de conversão.

word_censor($str, $censored[, $replacement = ''])¶
Parâmetros:
  • $str (string) – String de entrada
  • $censored (array) – Lista de palavras ruins para censurar.
  • $replacement (string) – O que usar para substituir as palavras ruins
Retorna:

String censurada
Tipo de retorno:

string

Permite que você censure as palavras dentro de uma string de texto. O primeiro parâmetro deverá conter a string original. O segundo parâmetro deverá conter um array de palavras que você proíbe. O terceiro parâmetro (opcional) pode conter o valor que substituirá as palavras. Se não for especificado serão utilizados caracteres: ###

Exemplo:

$disallowed = array('darn', 'shucks', 'golly', 'phooey');
$string = word_censor($string, $disallowed, 'Beep!');
highlight_code($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

String com código destacado (highlight) em HTML
Tipo de retorno:

string

Colore uma string do código (PHP, HTML, etc.). Exemplo:

$string = highlight_code($string);

Essa função usa a função highlight_string() do PHP. Logo, as cores utilizadas são as mesmas especificadas em seu arquivo php.ini .

highlight_phrase($str, $phrase[, $tag_open = '<mark>'[, $tag_close = '</mark>']])¶
Parâmetros:
  • $str (string) – String de entrada
  • $phrase (string) – Frase para destacar
  • $tag_open (string) – Tag de entrada usada no destaque
  • $tag_close (string) – Tag de fechamento usada no destaque
Retorna:

String com uma frase destacada em HTML
Tipo de retorno:

string

Destacará uma frase dentro de uma string de texto. O primeiro parâmetro deverá conter a string original. O segundo deverá conter a frase a ser destacada. Os terceiro e quarto parâmetros deverão conter as tags de abertura e fechamento do HTML que você deseja para envolver a frase.

Exemplo:

$string = "Aqui tem um bom texto string sobre nada em particular.";
echo highlight_phrase($string, "bom texto", '<span style="color:#990000;">', '</span>');

O código acima imprime:

Aqui tem um <span style="color:#990000;">bom texto</span> string sobre nada em particular.

Nota

Essa função costumava usar a tag <strong> por padrão. Navegadores antigos podem não suportar a nova tag de marcação do HTML5, então é recomendado que você insira o seguinte código CSS em sua folha de estilos se você precisar oferecer suporte a esses navegadores:

mark {
        background: #ff0;
        color: #000;
};
word_wrap($str[, $charlim = 76])¶
Parâmetros:
  • $str (string) – String de entrada
  • $charlim (int) – Limite de caracter
Retorna:

String com as palavras embrulhadas
Tipo de retorno:

string

Embrulhe texto em um número de caracter específico enquanto mantém as palavras completas.

Exemplo:

$string = "Aqui é uma simples string de texto que nos ajuda a demonstrar essa função.";
echo word_wrap($string, 25);

// Produziria:
// Aqui é uma simples
// string de texto que nos
// ajuda a demonstrar essa
// função.
ellipsize($str, $max_length[, $position = 1[, $ellipsis = '&hellip;']])¶
Parâmetros:
  • $str (string) – String de entrada
  • $max_length (int) – Tamanho limite da String
  • $position (mixed) – Posição para dividir em (int or float)
  • $ellipsis (string) – O que usar como caracter de reticência
Retorna:

String com reticência
Tipo de retorno:

string

Essa função irá tirar as tags de uma string, dividí-la em um tamanho máximo definido e inserir uma reticência.

O primeiro parâmetro é a string que receberá a reticência, o segundo é o número de caracteres no final da string. O terceiro parâmetro é onde na string a reticência será inserida, sendo 0 - 1, esquerda para a direita. Por exemplo, o valor 1 colocará a reticência à direita da string, .5 no meio, e 0 à esquerda.

Um quarto parâmetro opcional é o tipo da reticência. Por padrão, &hellip; será inserido.

Exemplo:

$str = 'this_string_is_entirely_too_long_and_might_break_my_design.jpg';
echo ellipsize($str, 32, .5);

Produz:

this_string_is_e&hellip;ak_my_design.jpg
