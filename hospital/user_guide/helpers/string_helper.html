Auxiliar de String¶

O arquivo Auxiliar de String contém funções que permitem trabalhar com cadeia de caracteres (strings).

Importante

Por favor, note que essas funções NÃO são destinadas, nem são adequadas para qualquer tipo de lógica relacionada à segurança.

  • Carregando este Auxiliar
  • Funções disponíveis

Carregando este Auxiliar¶

Este auxiliar é carregado usando o seguinte código:

$this->load->helper('string');

Funções disponíveis¶

As seguinte funções estão disponíveis:

random_string([$type = 'alnum'[, $len = 8]])¶
Parâmetros:
  • $type (string) – Tipo de randomização
  • $len (int) – Tamanho da saída da string
Retorna:

Uma string aleatória
Tipo de retorno:

string

Gera uma string aleatória com base no tipo e tamanho que você especifica. Útil para criar senhas ou gerar hashes aleatórias.

O primeira parâmetro especifica o tipo da string, o segunda parâmetro especifica o tamanhp. As seguintes opções estão disponíveis:

  • alpha: Uma string apenas com letras maiúsculas e minúsculas.
  • alnum: Uma string alfanumérica com caracteres maiúsculas e minúsculas.
  • basic: Um número aleatório com base no mt_rand().
  • numeric: Uma string de números.
  • nozero: Uma string sem zeros.
  • md5: Um número aleatório criptografado com base no md5() (comprimento fixo de 32).
  • sha1: Um número aleatório criptografado com base no sha1() (comprimento fixo de 40).

Exemplo de uso:

echo random_string('alnum', 16);

Nota

O uso dos tipos únicos e criptografado é DEPRECADO. Eles são apenas atalhos para md5 e sha1, respectivamente.

increment_string($str[, $separator = '_'[, $first = 1]])¶
Parâmetros:
  • $str (string) – String de entrada
  • $separator (string) – Separador para anexar com um número duplicado
  • $first (int) – Número inicial
Retorna:

Uma string incrementada
Tipo de retorno:

string

Incrementa uma string acrescentando um número ou aumentando o número. Útil para criar "cópias" ou um arquivo ou conteúdo de banco de dados duplicado que possui títulos únicos ou slugs.

Exemplo de uso:

echo increment_string('file', '_'); // "file_1"
echo increment_string('file', '-', 2); // "file-2"
echo increment_string('file_4'); // "file_5"
alternator($args)¶
Parâmetros:
  • $args (mixed) – Um número variável de argumentos
Retorna:

String(s) alternada(s)
Tipo de retorno:

mixed

Permite que dois ou mais itens sejam alternados entre si, quando estiver dentro de um loop. Exemplo:

for ($i = 0; $i < 10; $i++)
{
        echo alternator('string one', 'string two');
}

Você pode adicionar tantos parâmetros quanto quiser, e com cada iteração do loop o próximo item será retornado.

for ($i = 0; $i < 10; $i++)
{
        echo alternator('one', 'two', 'three', 'four', 'five');
}

Nota

Para usar várias chamadas separadas para esta função, basta chamar a função sem argumentos para reinicializar.

repeater($data[, $num = 1])¶
Parâmetros:
  • $data (string) – Entrada
  • $num (int) – Número de vezes para repetir
Retorna:

String repetida
Tipo de retorno:

string

Gera cópias repetidas dos dados que você envia. Exemplo:

$string = "\n";
echo repeater($string, 30);

O exemplo anterior geraria 30 novas linhas.

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() em vez disso.

reduce_double_slashes($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

Uma string com barras diagonais normalizadas
Tipo de retorno:

string

Converte barras duplas em uma seqüência de caracteres para uma barra única, exceto aquelas encontradas nos prefixos de protocolo de URL (p.e. http&#58;//).

Exemplo:

$string = "http://example.com//index.php";
echo reduce_double_slashes($string); // resulta em "http://example.com/index.php"
strip_slashes($data)¶
Parâmetros:
  • $data (mixed) – String de entrada ou um array de strings
Retorna:

String(s) com as barras removidas
Tipo de retorno:

mixed

Remove todas as barras de um array de strings.

Exemplo:

$str = array(
        'question'  => 'O seu nome é O\'reilly?',
        'answer' => 'Não, meu nome é O\'connor.'
);

$str = strip_slashes($str);

The above will return the following array:

array(
        'question'  => "O seu nome é O'reilly?",
        'answer' => "Não, meu nome é O'connor."
);

Nota

Por motivos históricos, esta função também aceita e manipula entradas de string. Isso, no entanto, torna apenas um apelido para stripslashes().

trim_slashes($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

String sem barras no início ou final
Tipo de retorno:

string

Remove todas barras no início e fim de uma string. Exemplo:

$string = "/this/that/theother/";
echo trim_slashes($string); // resulta em this/that/theother

Nota

Esta função é DEPRECADA. Use a função nativa trim() como alternativa: | | trim($str, ‘/’);

reduce_multiples($str[, $character = ''[, $trim = FALSE]])¶
Parâmetros:
  • $str (string) – Texto para pesquisar em
  • $character (string) – Caractere para reduzir
  • $trim (bool) – Se também cortar o caractere especificado
Retorna:

String reduzida
Tipo de retorno:

string

Reduz multiplas intâncias de um caractere em particular que ocorrem diretamente após a outra. Exemplo:

$string = "Fred, Bill,, Joe, Jimmy";
$string = reduce_multiples($string,","); //resulta em "Fred, Bill, Joe, Jimmy"

Se o terceiro parâmetro estiver definido como TRUE, ele irá remover as ocorrências do caractere no início e no final da string. Exemplo:

$string = ",Fred, Bill,, Joe, Jimmy,";
$string = reduce_multiples($string, ", ", TRUE); //resulta em "Fred, Bill, Joe, Jimmy"
quotes_to_entities($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

String com aspas convertidas em entidades HTML
Tipo de retorno:

string

Converte aspas simples e duplas em uma string para as entidades HTML correspondentes. Exemplo:

$string = "Joe's \"dinner\"";
$string = quotes_to_entities($string); //resulta em "Joe&#39;s &quot;dinner&quot;"
strip_quotes($str)¶
Parâmetros:
  • $str (string) – String de entrada
Retorna:

String com aspas espalhadas
Tipo de retorno:

string

Remove aspas simples e duplas de uma string. Exemplo:

$string = "Joe's \"dinner\"";
$string = strip_quotes($string); //resulta em "Joes dinner"
