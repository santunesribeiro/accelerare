Funções de Compatibilidade¶

CodeIgniter fornece um conjunto de funções de compatibilidade que permitem usar funções que de outra forma estão disponíveis de forma nativa em PHP, mas somente em versões superiores ou dependendo de uma determinada extensão.

Sendo implementações personalizadas, essas funções também terão algum conjunto de dependências por conta própria, mas ainda são úteis se sua configuração do PHP não as oferecer de forma nativa.

Nota

Muito parecido com funções comuns <common_functions>, as funções de compatibilidade estão sempre disponíveis, desde que suas dependências sejam atendidas.

Hashing de Senha¶

Este conjunto de funções de compatibilidade oferece um “backport” padrão da Extensão de Hashing de Senha do PHP que está disponível de outra forma somente desde o PHP 5.5.

Dependências¶

  • PHP 5.3.7
  • CRYPT_BLOWFISH suporte para crypt()

Constantes¶

  • PASSWORD_BCRYPT
  • PASSWORD_DEFAULT

Referência de Função¶

password_get_info($hash)¶
Parâmetros:
  • $hash (string) – Hash de senha
Retorna:

Informação sobre o hash da senha
Tipo de retorno:

matriz

Para obter mais informações, consulte o manual do PHP para password_get_info().

password_hash($password, $algo[, $options = array()])¶
Parâmetros:
  • $password (string) – Senha de texto simples
  • $algo (int) – Algoritmo Hashing
  • $options (array) – Opções de Hashing
Retorna:

Senha Hashed ou FALSE em caso de falha
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para password_hash().

Nota

A menos que você forneça seu próprio salt (e válido), esta função possui uma dependência adicional de uma fonte CSPRNG disponível. Cada um dos seguintes satisfaz isso: - mcrypt_create_iv() com MCRYPT_DEV_URANDOM - openssl_random_pseudo_bytes() - /dev/arandom - /dev/urandom

password_needs_rehash()¶
Parâmetros:
  • $hash (string) – Hash de senha
  • $algo (int) – Algorítimo de Hashing
  • $options (array) – Opções de Hashing
Retorna:

VERDADEIRO se o hash deve ser devolvido para coincidir com o algoritmo e opções fornecidos, FALSO caso contrário.
Tipo de retorno:

booleano

Para obter mais informações, consulte o manual PHP para password_needs_rehash().

password_verify($password, $hash)¶
Parâmetros:
  • $password (string) – Senha de texto simples
  • $hash (string) – Hash de senha
Retorna:

VERDADEIRO se a senha corresponder ao hash, FALSO se não
Tipo de retorno:

booleano

Para obter mais informações, consulte o manual PHP para password_verify().

Hash (Mensagem Digest)¶

Esta camada de compatibilidade contém backports para as funções hash_equals() e hash_pbkdf2() que de outra forma requerem PHP 5.6 e / ou PHP 5.5 respectivamente

Dependências¶

  • Nenhuma

Referência de Funções¶

hash_equals($known_string, $user_string)¶
Parâmetros:
  • $known_string (string) – Conjunto de caracteres conhecidos
  • $user_string (string) – Conjunto fornecida pelo usuário
Retorna:

VERDADEIRO se as sequências de caracteres combinarem, FALSO caso contrário
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para hash_equals().

hash_pbkdf2($algo, $password, $salt, $iterations[, $length = 0[, $raw_output = FALSE]])¶
Parâmetros:
  • $algo (string) – Algorítimo de Hashing
  • $password (string) – Senha
  • $salt (string) – Hash salt
  • $iterations (int) – Número de iterações a serem executadas durante a derivação
  • $length (int) – Tamanho do conjunto de caracteres de saída
  • $raw_output (bool) – Quer retornar dados binários
Retorna:

Chave derivada de senha ou FALSE em caso de falha
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para hash_pbkdf2().

Conjunto de caracteres Multibyte¶

Este conjunto de funções de compatibilidade oferece suporte limitado para a extensão Extensão do conjunto de caracteres Multibyte. Devido às soluções alternativas limitadas, apenas algumas funções estão disponíveis.

Nota

Quando um parâmetro de conjunto de caracteres é omitido, $config['charset'] será usado.

Dependências¶

  • iconv extensão

Importante

Essa dependência é opcional e essas funções sempre serão declaradas. Se o iconv não estiver disponível, eles retornarão às versões de não-mbstring.

Importante

Quando um conjunto de caracteres é fornecido, ele deve ser suportado por iconv e em um formato que ele reconheça.

Nota

Para sua própria verificação de dependência na extensão real da mbstring, use a constante MB_ENABLED.

Referência de Função¶

mb_strlen($str[, $encoding = NULL])¶
Parâmetros:
  • $str (string) – Conjunto de caracteres de entrada
  • $encoding (string) – Conjunto de caracteres configurado
Retorna:

Número de caracteres no conjunto de caracteres de entrada ou FALSO em caso de falha
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para mb_strlen().

mb_strpos($haystack, $needle[, $offset = 0[, $encoding = NULL]])¶
Parâmetros:
  • $haystack (string) – Conjunto de caracteres para pesquisar em
  • $needle (string) – Parte do conjunto de caracteres para procurar
  • $offset (int) – Pesquisa deslocada
  • $encoding (string) – Conjunto de caracteres
Retorna:

Posição do caractere numérico de onde $needle foi encontrada ou FALSO se não encontrada
Tipo de retorno:

vários tipos

Para obter mais informações, consulte o manual PHP para mb_strpos().

mb_substr($str, $start[, $length = NULL[, $encoding = NULL]])¶
Parâmetros:
  • $str (string) – Conjunto de caracteres de entrada
  • $start (int) – Posição do primeiro caracter
  • $length (int) – Número máximo de caracteres
  • $encoding (string) – Conjunto de caracteres
Retorna:

Porção da $str especificada de $start e $length ou FALSE em caso de falha.
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para mb_substr().

Funções padrão¶

Este conjunto de funções de compatibilidade oferece suporte para algumas funções padrão no PHP que, de outra forma, exigem uma versão mais recente do PHP.

Dependências¶

  • Nenhuma

Referência de Função¶

array_column(array $array, $column_key[, $index_key = NULL])¶
Parâmetros:
  • $array (array) – Matriz para buscar caracteres de
  • $column_key (mixed) – Chave da coluna para retornar valores de
  • $index_key (mixed) – Chave para usar para os valores retornados
Retorna:

Uma matriz de valores que representam uma única coluna da matriz de entrada
Tipo de retorno:

matriz

Para obter mais informações, consulte o manual PHP para array_column().

array_replace(array $array1[, ...])¶
Parâmetros:
  • $array1 (array) – Matriz para substituir elementos
  • ... (array) – Matriz (ou múltiplos) para extrair elementos
Retorna:

Matriz modificada
Tipo de retorno:

matriz

Para obter mais informações, consulte o manual PHP para array_replace().

array_replace_recursive(array $array1[, ...])¶
Parâmetros:
  • $array1 (array) – Matriz para substituir elementos
  • ... (array) –Matriz (ou múltiplos) para extrair elementos
Retorna:

Matriz modificada
Tipo de retorno:

matriz

Para obter mais informações, consulte o manual PHP para array_replace_recursive().

Importante

Somente a função nativa do PHP pode detectar recursões infinitas. A menos que você esteja executando o PHP 5.3, tenha cuidado com as referências!

hex2bin($data)¶
Parâmetros:
  • $data (array) – Representação hexadecimal de dados
Retorna:

Representação binária dos dados fornecidos
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para hex2bin().

quoted_printable_encode($str)¶
Parâmetros:
  • $str (string) – Conjunto de caractere de entrada
Retorna:

Conjunto de caracteres 8bit-encoded
Tipo de retorno:

conjunto de caracteres

Para obter mais informações, consulte o manual PHP para quoted_printable_encode().
