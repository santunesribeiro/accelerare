Gerando Resultados de Consulta¶

Existem várias maneiras de se gerar resultados de consultas:

  • Vários Resultados em um Array
  • Resultado Único
  • Resultados Personalizados como Objetos
  • Funções Auxiliares de Resultados
  • Referência da Classe

Vários Resultados em um Array¶

result()

Este método retorna o resultado da consulta como um array de objetos, ou um array vazio em caso de falha. Normalmente você vai usar este método em uma iteração com foreach, como esta:

$query = $this->db->query("SUA CONSULTA");
  
  foreach ($query->result() as $row)
  {
          echo $row->title;
          echo $row->name;
          echo $row->body;
  }

O método acima é um atalho para result_object().

Você também pode passar uma string para result() informando a classe a ser instanciada para o objeto resultante do foreach (observação: esta classe deve estar carregada)

$query = $this->db->query("SELECT * FROM users;");

foreach ($query->result('User') as $user)
{
        echo $user->name; // acessar atributos
        echo $user->reverse_name(); // ou métodos definidos na classe 'User'
}

result_array()

Este método retorna como resultado da consulta um array puro, ou um array vazio quando nenhum resultado é encontrado. Normalmente você vai usar isto em uma iteração através do foreach, como esta:

$query = $this->db->query("SUA CONSULTA");

foreach ($query->result_array() as $row)
{
        echo $row['title'];
        echo $row['name'];
        echo $row['body'];
}

Resultado Único¶

row()

Este método retorna uma linha em um único resultado. Se sua consulta tiver mais que um resultado, irá retornar a primeira linha. O resultado é retornado como um objeto. Aqui está um exemplo de uso:

$query = $this->db->query("SUA CONSULTA");

$row = $query->row();

if (isset($row))
{
        echo $row->title;
        echo $row->name;
        echo $row->body;
}

Se você quer uma linha específica retornada, você pode informar o número da linha como um digito no primeiro parâmetro:

$row = $query->row(5);

Você também pode adicionar uma String como segundo parâmetro, que é o nome da classe em que a linha deverá ser instanciada:

$query = $this->db->query("SELECT * FROM users LIMIT 1;");
$row = $query->row(0, 'User');

echo $row->name; // access attributes
echo $row->reverse_name(); // ou métoos definidos para a classe 'User'

row_array()

Idêntico ao método row() acima, exceto que o resultado é retornado como um array. Exemplo:

$query = $this->db->query("SUA CONSULTA");

$row = $query->row_array();

if (isset($row))
{
        echo $row['title'];
        echo $row['name'];
        echo $row['body'];
}

Se você quer uma linha específica retornada, você pode informar o número da linha como um dígito no primeiro parâmetro:

$row = $query->row_array(5);

Além disso, você pode navegar (para frente/para trás/primeiro/último) nos seus resultados utilizando estas variações:

$row = $query->first_row() // primeira linha
$row = $query->last_row() // última linha
$row = $query->next_row() // próxima linha
$row = $query->previous_row() // linha anterior

Por padrão retorna um objeto a menos que você informa a palavra “array” no parâmetro:

$row = $query->first_row(‘array’)
$row = $query->last_row(‘array’)
$row = $query->next_row(‘array’)
$row = $query->previous_row(‘array’)

Nota

Todos os métodos acima vão carregar todos os resultados na memória (pré-processamento). Use unbuffered_row() para processamento de grandes quantidades de registros.

unbuffered_row()

Este método retorna uma única linha como resultado, sem pré-processar todo resultado na memória como o método row() faz. Se sua query tem mais do que uma linha de resultado, irá retornar a linha atual e mover o ponteiro interno de dados para frente.

$query = $this->db->query("SUA CONSULTA");

while ($row = $query->unbuffered_row())
{
        echo $row->title;
        echo $row->name;
        echo $row->body;
}

Você pode opcionalmente passar ‘object’ (padrão) ou ‘array’ como parâmetro para especificar o tipo do valor retornado:

$query->unbuffered_row();               // objeto
$query->unbuffered_row('object');       // objeto
$query->unbuffered_row('array');        // array associativo

Resultado Como Objetos Personalizados¶

Você pode ter os resultados retornados como um instância de uma classe ao invés de uma stdClass ou array, como os métodos result() e result_array() permitem. Isto requer que a classe já esteja carregada na memória. O objeto vai ter todos os valores retornados do banco de dados definidos como propriedades. Se eles já foram declarados e não são públicos, então você deve providenciar um método __set() para permitir que eles sejam definidos.

Exemplo:

class User {

        public $id;
        public $email;
        public $username;

        protected $last_login;

        public function last_login($format)
        {
                return $this->last_login->format($format);
        }

        public function __set($name, $value)
        {
                if ($name === 'last_login')
                {
                        $this->last_login = DateTime::createFromFormat('U', $value);
                }
        }

        public function __get($name)
        {
                if (isset($this->$name))
                {
                        return $this->$name;
                }
        }
}

Além dos dois métodos listados acima, os seguintes métodos também podem ter um nome de classe para retornar o resultado como: first_row(), last_row(), next_row(), e previous_row().

custom_result_object()

Retorna todos os resultados como um array de instâncias de uma classe requerida. O único parâmetro é o nome da classe a ser instanciada.

Exemplo:

$query = $this->db->query("SUA CONSULTA");

$rows = $query->custom_result_object('User');

foreach ($rows as $row)
{
        echo $row->id;
        echo $row->email;
        echo $row->last_login('Y-m-d');
}

custom_row_object()

Retorna um único registro dos resultados de sua consulta/query. O primeiro parâmetro é o número da linha do resultado. O segundo parâmetro é o nome da classe a ser instanciada.

Exemplo:

$query = $this->db->query("SUA CONSULTA");

$row = $query->custom_row_object(0, 'User');

if (isset($row))
{
        echo $row->email;   // access attributes
        echo $row->last_login('Y-m-d');   // access class methods
}

Você também pode usar o métodorow() exatamente do mesmo jeito.

Exemplo:

$row = $query->custom_row_object(0, 'User');

Funções Auxiliares de Resultados¶

num_rows()

O número de linhas retornado pela consulta/query. Nota: Neste exemplo, $query é a variável ao qual o resultado da consulta está associado:

$query = $this->db->query('SELECT * FROM my_table');

echo $query->num_rows();

Nota

Nem todos os drivers de banco possuem uma forma nativa de se obter o número total de linhas de um conjunto de resultados. Quando este é o caso, todos os dados são pré-processados e o método count() é manualmente chamado no array resultante em ordem para alcançar o mesmo resultado.

num_fields()

O número de CAMPOS (colunas) retornado pela consulta/query. Tenha certeza de chamar o método usando o objeto resultante de sua consulta:

$query = $this->db->query('SELECT * FROM my_table');

echo $query->num_fields();

free_result()

Libera a memória associada com o resultado e deleta o ID do recurso resultante. Normalmente o PHP libera sua memória automaticamente no fim da execução do script. Todavia, se você está executando um conjunto de consultas/queries em um script particular você deve querer liberar o resultado depois de cada resultado de consulta ter sido gerado em ordem para cortar o consumo de memória.

Exemplo:

$query = $this->db->query('SELECT title FROM my_table');

foreach ($query->result() as $row)
{
        echo $row->title;
}

$query->free_result();  // O objeto resultado $query  não estará mais disponível

$query2 = $this->db->query('SELECT name FROM some_table');

$row = $query2->row();
echo $row->name;
$query2->free_result(); // O objeto resultado $query2  não estará mais disponível

data_seek()

Este método define o ponteiro interno para a próxima linha para ser buscado. Ele só é útil em combinação com unbuffered_row().

Ele aceita um valor inteiro positivo, cujo padrão é 0 e retorna TRUE em caso de sucesso ou FALSO em caso de falha.

$query = $this->db->query('SELECT `field_name` FROM `table_name`');
$query->data_seek(5); // Pular as 5 primeiras linhas
$row = $query->unbuffered_row();

Nota

Nem todos os drivers de banco de dados suportam este recurso e vão retornar FALSE. Mais notavelmente - você não poderá usá-lo com PDO.
